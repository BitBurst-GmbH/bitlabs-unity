# Changelog

All notable changes to this package will be documented in this file.

## [4.0.0] - 2024-11-05

### Added

- UPM package structure
- Assembly definitions for faster compilation
- EDM4U automatic dependency resolution
- Modern callback-based API
- Android and iOS native SDK integration
- Survey integration and reward callbacks

### Changed

- Migrated to Packages folder structure
- Replaced UnitySendMessage with callback-based architecture

### Fixed

- iOS Swift bridge integration
- Android JNI callback handling
- IL2CPP compatibility
